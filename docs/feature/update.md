# Update

## 개요

기존 시료 데이터를 선택하여 특정 필드를 수정하는 기능.

## 동작

- 사용자로부터 수정할 대상의 ID를 입력받아 [read](read.md) 검색 기능으로 대상을 찾는다.
- 대상이 존재하지 않으면 수정 불가를 알리고(`false` 반환) 종료한다 — 예외를 던지지 않는다.
- 대상이 존재하면 수정할 필드와 새 값을 입력받는다.
- 새 값에 대해 형식/필수 항목 검증을 수행한다(Create와 동일한 검증 규칙 적용).
- 검증을 통과하면 필드를 반영하는 변경 로직(`std::function<void(Sample&)>` mutator)을 적용하기 전에 현재 상태를 백업해둔다.
- mutator를 적용해 메모리 상의 해당 `Sample`을 갱신한다.
- [json-file-storage](json-file-storage.md)를 호출해 변경된 목록을 JSON 파일에 저장한다.
- 저장이 실패하면 백업해둔 값으로 되돌리고(롤백), 예외를 상위로 전달한다.
- 성공 시 `true`를 반환한다.

## 관련 기능

- [read](read.md): 수정 대상을 찾을 때 사용.
- [json-file-storage](json-file-storage.md): 수정 결과를 파일에 반영.
